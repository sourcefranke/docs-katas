# Docs Katas

## CLI Commands

### Install mkdocs

````
pip install mkdocs
pip install mkdocs-mermaid2-plugin
````

### Initialize new project

````
python -m mkdocs new <name>
````

### Run static web page locally

````
python -m mkdocs serve
````

### Deploy at GitHub Pages

````
python -m mkdocs gh-deploy
````
